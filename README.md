## 📜 Shell Scripts Overview

### 🔹 Script 1: System Identity Report  
Displays system information such as kernel version, user details, uptime, OS information, and system license.

### 🔹 Script 2: FOSS Package Inspector  
Checks whether Git is installed and displays its version and basic details along with a short description.

### 🔹 Script 3: Disk and Permission Auditor  
Analyzes important system directories and shows permissions, ownership, and storage usage.

### 🔹 Script 4: Log File Analyzer  
Reads log files, searches for specific keywords, counts occurrences, and displays recent matching entries.

### 🔹 Script 5: Open Source Manifesto Generator  
Generates a personalized open-source philosophy statement based on user input and saves it to a file.
